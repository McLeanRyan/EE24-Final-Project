## EE24 Final Project: Hurricane Simulation
Ryan McLean, Landon Meng, Lauren O'Rourke

This repository contains the code and data set necessary to produce the results
shown in our presentation. To access the data set, decompress the zip file 
ibtracs.NA.list.v04r01.csv.zip. Running the MATLAB script ee24_finalproject.m
will produce the results of our path prediction model for Hurricane Sandy, and
can be modified to predict the path of other named storms if desired.
